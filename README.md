CommonUtil项目简介
-----
是一系列通用类、辅助类、工具类的集合，有以下特点：
- **1. 通用性强**：只有常用、通用才集入。
- **2. 体积超小**：不到50K！加入增强IO包混淆后70K！
- **3. 纯  纯  纯**：类间独立，单挑大梁，极少耦合，就是单纯！


 1.utils包：常用工具类
-----
- **AndroidUtil**：     android信息， 获取android手机品牌、商家、版本号等信息
- **AppUtil**：         app工具， 检测是否前台运行
